Ejemplo: 'class' y 'v-bind:class'

La clase 'impClass' se establece de manera normal con el atributo de clase, y la clase 'yelClass' se establece con v-bind:class. El resultado es que Vue fusiona las clases para que el elemento div pertenezca a ambas clases.

Este div pertenece a las clases 'impClass' y 'yelClass'.
